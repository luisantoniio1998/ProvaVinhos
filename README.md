# WineTasting

Android app for wine shop inventory management using Kotlin and local database.

## Technologies
- Kotlin
- Android Studio
- SQLite Database
- Android UI Components

## Features
- Stock management
- Sales tracking
- Inventory control
- Local database operations
- User authentication

## Setup
1. Android Studio installed
2. Clone repository
3. Build project
4. Run on Android device/emulator

## Requirements
- Android 6.0+
- Android Studio
- JDK 8+
